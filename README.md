<h1>KIRAR SHOES 👟</h1>

<h4>
Modern ve kullanıcı dostu bir e-ticaret web sitesi projesidir. 
Bu projede HTML5, CSS3 ve JavaScript kullanılarak kullanıcı giriş sistemi, 
ürün listeleme, sepet işlemleri, favoriler, sipariş oluşturma ve admin paneli gibi 
özelliklere sahip gelişmiş bir alışveriş sitesi geliştirilmiştir.
</h4>

<h1>Kullanılan Teknolojiler</h1>

<h4>
HTML5 - CSS3 - JavaScript - LocalStorage
</h4>
